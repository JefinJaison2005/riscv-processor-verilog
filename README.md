# riscv-processor-verilog
# 32-bit RISC-V (RV32I) Processor using Verilog

This project implements a 32-bit RISC-V (RV32I) processor using Verilog in Xilinx Vivado.

## 🚀 Overview
Designed as part of a hackathon, this processor supports multiple instruction formats and integrates key components required for execution.

## ⚙️ Features
- Supports R, I, S, B, U, and J-type instructions
- Modular design with separate functional units
- Branching and jumping support
- Memory operations (Load/Store)
- ALU supporting arithmetic, logical, and shift operations

## 🧩 Modules
- Program Counter  
- Instruction Memory  
- Register File  
- Immediate Generator  
- ALU  
- Control Unit  
- ALU Control  
- Data Memory  
- Branch Unit  

## 🛠️ Tools Used
- Verilog HDL  
- Xilinx Vivado  
- RISC-V Architecture  

## ▶️ Working
1. Fetch instruction  
2. Decode instruction  
3. Execute operation  
4. Memory access  
5. Write back result  

## 📊 Results
- Successfully simulated in Vivado  
- Verified instruction execution
